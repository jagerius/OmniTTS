# OmniTTS для SkyrimNet

## Опис
OmniTTS — це високопродуктивний модуль перетворення тексту в мовлення (TTS), який замінює стандартний рушій Chatterbox у SkyrimNet. Інтегрує найсучаснішу дифузійну модель OmniVoice для забезпечення реалістичного клонування голосу та динамічної генерації діалогів для NPC у Skyrim. Уся генерація проводиться локально за допомогою серверного бекенду Gradio, який на 100% сумісний із C++ клієнтом SkyrimNet.

## Встановлення
1. Перейдіть до каталогу `OmniTTS` всередині папки встановлення SkyrimNet.
2. Запустіть скрипт `setup_venv.bat` (Windows). Він автоматично створить віртуальне середовище Python і завантажить усі потрібні компоненти, разом з PyTorch (з підтримкою CUDA-прискорення).
3. Переконайтеся, що ви додали короткі голосові зразки `.wav` у директорію `speakers/`, інакше клонування персонажів буде неможливим.

> [!NOTE]
> За замовчуванням для роботи OmniTTS потрібна відеокарта NVIDIA (використовується прискорення CUDA). Якщо у вас немає сумісної відеокарти, ви можете примусово запустити генерацію на процесорі (CPU), встановивши `use_cpu = true` у файлі `omnitts_config.txt` (увага: генерація буде значно повільнішою).

## Використання
1. Для запуску TTS сервера відкрийте файл `Start.bat`. У зв'язку з цим запуститься локальний ендпоінт Gradio, що працює на порту 8000.
2. Сервер очікуватиме і оброблятиме API запити безпосередньо від SkyrimNet GamePlugin.
3. Щоразу, коли ви взаємодієте з NPC в грі, SkyrimNet самостійно надсилатиме серверу потрібний текст, а OmniTTS швидко згенерує діалог і поверне готове аудіо.

## Налаштування (`omnitts_config.txt`)
Файл конфігурації `omnitts_config.txt` дозволяє детальніше підлаштувати параметри генерації:
* **`guidance_scale`**: Визначає наскільки точно генерація має відповідати голосовому інпуту (промпту) і тексту.
* **`num_step`**: Кількість кроків інференсу в дифузійній моделі. Зменшення (наприклад 12 замість 32) призведе до дуже сильного пришвидшення і слабкого погіршення якості.
* **`speed`**: Ступінь швидкості мовлення розмовника.
* **`denoise`**: Вмикає приглушення й усунення шумів із генеруємого аудіо. Краще залишати на `true`.
* **`audio_chunk_threshold` & `audio_chunk_duration`**: Якщо діалоги задовгі ці додаткові фічі ділять їх на уривки для запобігання заїкання та обрізання слів.
* **`model_path`**: Вказує локальний шлях, або адресу на HuggingFace для базової моделі OmniVoice.
* **`enable_memory_cache` & `enable_disk_cache`**: Кешування клонованого голосу дуже пришвидшить отримання відповідей при багаторазовому спілкуванні зі знайомим персонажем.
