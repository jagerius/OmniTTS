# SkyrimNet OmniTTS

## 概要 (Description)
OmniTTS は、SkyrimNet のデフォルト Chatterbox エンジンを置き換える高性能な音声合成 (TTS) モジュールです。最先端の拡散モデルをベースとした OmniVoice TTS をネイティブに統合し、スカイリム (Skyrim) の NPC 向けに、リアルでダイナミックな音声クローンとテキストからの対話生成を提供します。すべての生成作業はローカルの Gradio バックエンドを通じて実行され、SkyrimNet の C++ クライアントとシームレスに互換性があります。

## インストール手順 (Installation)
1. SkyrimNet 内の `OmniTTS` ディレクトリに移動してください。
2. Windows環境で `setup_venv.bat` を実行します。このスクリプトは、Python仮想環境を自動作成し、CUDA 互換 PyTorch を含む必要なすべての依存関係をインストールします。
3. キャラクターの音声をクローンするためには、その音声サンプルとなる `.wav` ファイルを `speakers/` ディレクトリ内に配置されていることを確認してください。

## 使い方 (Usage)
1. TTS サーバーを起動するために、`Start.bat` を実行します。ポート8000番でローカル Gradio エンドポイントが起動します。
2. 起動したサーバーは SkyrimNet GamePlugin から送られてくる API リクエストを待機し続けます。
3. ゲーム内で NPC とやり取りを行うと、SkyrimNet が自動でテキストデータを送信し、OmniTTS がセリフの音声を生成した上で音声のファイルパスをゲーム側に返します。

## 設定ファイル (`omnitts_config.txt`)
`omnitts_config.txt` ファイルを通して、生成の質やパフォーマンスを微調整することができます。変更可能な主なパラメータは以下の通りです：
* **`guidance_scale`**: AIが参照元となる音声波形と入力テキストに対してどれくらい厳密に従うかを定義します。
* **`num_step`**: 拡散モデルの推論ステップ数です。値を下げる（例: 12）ことで、音質のわずかな低下と引き換えに生成速度を大幅に向上させます。
* **`speed`**: NPC の話すスピード（テンポ）を制御します。
* **`denoise`**: よりクリアな音声出力のためにノイズ除去トークンを有効にします（`true`のままにしておくことを推奨します）。
* **`audio_chunk_threshold` & `audio_chunk_duration`**: 長い文やパラグラフの生成中に音声が途切れるのを防ぐため、長い対話データを処理可能な小さな音声チャンクに分割します。
* **`model_path`**: OmniVoice モデルの HuggingFace リポジトリ名、またはローカルの参照パスを指定します。
* **`enable_memory_cache` & `enable_disk_cache`**: 同じ NPC と繰り返し会話する際にプロンプト属性をキャッシュし、レスポンス時間を劇的に短縮させます。
