# OmniTTS для SkyrimNet

## Описание
OmniTTS — это высокопроизводительный модуль преобразования текста в речь (TTS), заменяющий стандартный движок Chatterbox в SkyrimNet. Он интегрирует передовую диффузионную модель OmniVoice для создания реалистичного клонирования голоса и динамической генерации диалогов для NPC в Skyrim. Вся обработка выполняется локально через бэкенд Gradio, который полностью совместим с C++ клиентом SkyrimNet.

## Установка
1. Перейдите в каталог `OmniTTS` внутри вашей установки SkyrimNet.
2. Запустите `setup_venv.bat` (Windows). Этот скрипт автоматически создаст виртуальную среду Python и установит все необходимые зависимости, включая PyTorch с поддержкой CUDA.
3. Убедитесь, что в директории `speakers/` размещены файлы образцов голоса `.wav` для клонирования персонажей.
4. **Важно для пользователей Windows**: OmniVoice требует библиотеку `torchcodec`, которая зависит от полной установки FFmpeg.
   - Загрузите полную (FULL) сборку FFmpeg с [BtBN FFmpeg-Builds](https://github.com/BtbN/FFmpeg-Builds/releases).
   - Распакуйте архив и переименуйте папку в `ffmpeg`.
   - Поместите эту папку `ffmpeg` прямо в корневую директорию (`SkyrimNetOmniVoice`) или внутри `OmniTTS`, так чтобы существовал путь `ffmpeg/bin`. OmniTTS автоматически обнаружит и загрузит ее для предотвращения сбоев.

> [!NOTE]
> По умолчанию для работы OmniTTS требуется видеокарта NVIDIA (используется ускорение CUDA). Если у вас нет совместимой видеокарти, вы можете принудительно запустить генерацию на процессоре (CPU), установив `use_cpu = true` в файле `omnitts_config.txt` (внимание: генерация будет значительно медленнее).

## Использование
1. Для запуска TTS-сервера выполните `Start.bat`. Это запустит локальный сервер Gradio на порту 8000.
2. Сервер будет активно отслеживать API-запросы от плагина SkyrimNet GamePlugin.
3. Во время взаимодействия с NPC в Skyrim, SkyrimNet отправляет текст, а OmniTTS генерирует диалог и возвращает путь к аудио.

## Конфигурация (`omnitts_config.txt`)
Файл `omnitts_config.txt` позволяет тонко настраивать параметры генерации:
* **`guidance_scale`**: Определяет, насколько точно ИИ придерживается голосового шаблона и текста.
* **`num_step`**: Количество шагов диффузионной рендеринговой модели. Снижение значения (например, до 12) значительно ускоряет генерацию при незначительных потерях качества.
* **`speed`**: Скорость речи персонажа.
* **`denoise`**: Включает алгоритм подавления шума в генерируемом аудио (рекомендуется `true`).
* **`audio_chunk_threshold` & `audio_chunk_duration`**: Предотвращает обрывы длинных предложений путем разделения обширных текстов на короткие, легко обрабатываемые фрагменты звука.
* **`model_path`**: Указывает репозиторий HuggingFace или локальный путь к модели OmniVoice.
* **`enable_memory_cache` & `enable_disk_cache`**: Кэширует параметры клонирования голоса, чтобы радикально ускорить генерацию при повторном разговоре с одним и тем же NPC.
